# Changelog

## [0.1.0] - 2026-03-12

### Added
- Voice recording via WebView panel with microphone capture
- Web Speech API transcription engine (free, no API key)
- OpenAI Whisper API transcription engine (higher accuracy)
- Toggle recording with `Ctrl+Shift+H` keyboard shortcut
- Start/Stop/Toggle recording commands in Command Palette
- Status bar indicator (idle/recording/processing)
- Clipboard output with auto-open Cursor chat
- Editor insert output mode
- Silence detection with configurable timeout
- Prompt cleanup (filler word removal, punctuation normalization)
- Voice commands (stop, new line, period, comma, etc.)
- Multi-language support (14 languages)
- Voice history with Quick Pick selector
- Real-time transcription preview in WebView panel
- Audio level meter visualization
